# Dummy Data Patterns

목업에 사용할 그럴듯한 한국어 더미 데이터.

## 인명 (한국어)
- 김민준, 이서연, 박지호, 최하은, 정우진, 강시우, 조유나, 윤도현, 장서아, 임준서

## 사업장명 (팜이지 맥락)
- 그린팜 수원점, 해피베리 농장, 청정농원, 산들바람 유기농, 햇살마루 과수원

## 지역명
- 서울특별시 강남구, 경기도 수원시 영통구, 부산광역시 해운대구, 제주특별자치도 서귀포시

## 날짜 (현실적인 범위)
- 최근 30일 기준, YYYY-MM-DD 형식
- 오래된 데이터는 1~2년 전

## 숫자 (금액, 수량)
- 금액: 천 단위 콤마, "원" 접미사 (e.g., 1,234,567원)
- 수량: 적절한 단위 (kg, 개, 포기 등)

## GIS/좌표 (SIGMA-AI 맥락)
- 서울 좌표: 37.5665, 126.9780
- 부산 좌표: 35.1796, 129.0756
- 제주 좌표: 33.4996, 126.5312
